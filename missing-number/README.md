# Missing Number

> About Problem for Live Coding

Given a integer numbers array, Find the missing number in the sequence from Numbers given

## Example 1

1. `N = 10`
2. `A = [0,5,3,7,9,1,2,8,6,10]`
   Output: 4

## Example 2

1. `N = 2`
2. `A = [0,2]`
   Output: 1

## Example 3

1. `N = 5`
2. `A = [0,1,2,4,3]`
   Output: 5

## Example 4

1. `N = 12`
2. `A = [12,0,4,3,7,11,9,1,5,8,6,10]`
   Output: 2

</p>

## Constraints

- Just Integer Number
- No repeat Numbers inside
